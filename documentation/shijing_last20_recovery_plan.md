# Shijing last-20 recovery plan

This plan tracks the remaining *Shijing* extant-poem recovery work after the `Finalize Shijing unresolved witness report` baseline. The current broken Internet Archive hOCR witness remains the blocker of record for the unresolved queue, but the recovery sprint now prefers cleaner public-domain fulltext mirrors, alternate scans, and documented manual-paste intake over forcing more exports from bad OCR.

## Witness search snapshot

Searches on 2026-05-31 checked these public-domain routes:

| Route | Status | Notes |
| --- | --- | --- |
| Internet Archive `chineseclassics41legg` hOCR/fulltext | Baseline checked | Existing blocker notes come from these page spans; keep as the current-witness baseline, not the preferred recovery route. |
| ChineseNotes Legge mirror (`alexamies/chinesenotes.com`) | Candidate found | Raw GitHub mirror files were reachable for all 20 unresolved extant poems and each file contained a non-empty English poem body. Four poems were clean enough to promote in this batch; `guofeng-zhengfeng-009` still needs an independent cross-check because the mirror contains the typo `ephermeral`. |
| Sacred Texts / SBE Vol. III | Automation blocked | Direct scripted fetch attempts hit access/403 limits, so treat this as a manual cross-check route when needed. |
| Chinese Text Project | Manual-assist route | Keep for manual paste or human cross-check when a mirror needs corroboration. |
| Chinese Text Initiative / UVA AnoShih | Unavailable | Historical links surfaced in search, but the old UVA route did not yield a usable live witness. |
| Wengu / other Legge mirrors | Secondary manual route | Useful as corroboration when a mirror looks promising but still needs a second public-domain witness. |
