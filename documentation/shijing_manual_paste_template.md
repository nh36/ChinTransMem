# Shijing manual paste template

Use this template when a public-domain witness is visible to a human reviewer but awkward to capture automatically. Save each local-only intake as:

`scratch/shijing_last20_manual_paste/{section_id}__{source_slug}__YYYY-MM-DD.md`

Paste English verse only. Chinese text is optional and should be included only when it helps confirm stanza boundaries or catch OCR/mirror corruption.

```md
# Shijing manual paste intake

- section_id:
- source_url:
- source_name:
- access_date:
- poem_title_as_displayed:
- witness_type: Legge / SBE / CTI / Wengu / other public-domain witness

## English text only

Paste only the English poem body here.
Do not include commentary, page headers, page numbers, Chinese quoted text, notes, or adjacent poems.

## Chinese text for cross-checking (optional)

Paste only if it helps confirm stanza boundaries or poem identity.

## Notes

- Is this a direct Legge witness, an SBE page, a CTI mirror, or another public-domain source?
- Did you notice any suspect wording, truncation, missing stanza breaks, or obvious OCR/mirror errors?
- If this paste resolves a previously blocked poem, note the old blocker and the page or anchor that fixes it.
```
